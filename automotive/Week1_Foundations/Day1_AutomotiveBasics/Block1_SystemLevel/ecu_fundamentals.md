# ECU Fundamentals

- ECU = embedded controller for a specific vehicle function  
- Cars have 70–100+ ECUs  
- Core parts: MCU, Flash, RAM, peripherals, comm interfaces  
- Golden flow: Sensor → ECU → Actuator  
- Real-time constraints (ms to µs)  
- ECU domains: Powertrain, Chassis, Body, Infotainment, ADAS
